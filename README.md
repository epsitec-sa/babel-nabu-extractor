# babel-nabu-extractor

Extract Nabu string with babel.

## Node

```js
var babelCore = require ('babelCore');

babelCore.transform (code, {
  plugins: ['babel-nabu-extractor']
});
```

## CLI

```sh
babel --plugins babel-nabu-extractor code.js
```
