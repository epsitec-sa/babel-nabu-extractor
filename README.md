# babel-plugin-nabu-extractor

Extract Nabu strings with babel.

## Node

```js
const babelCore = require ('babelCore');

babelCore.transform (code, {
  plugins: ['babel-plugin-nabu-extractor']
});
```

## CLI

```sh
babel --plugins babel-plugin-nabu-extractor code.js
```
